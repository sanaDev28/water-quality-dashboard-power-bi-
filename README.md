# water-quality-dashboard-power-bi-
Power BI dashboard for water quality analysis using data visualization and business intelligence techniques.
Smart Water Quality Monitoring Dashboard
Project Overview

This project analyzes water quality data using Microsoft Power BI and Python. The dashboard provides insights into water contamination, disease risks, and environmental conditions across different cities.

Objectives
Analyze water quality indicators
Identify critical disease-risk areas
Study contamination patterns
Visualize seasonal variations
Support environmental decision-making

Key Findings
Chennai and New Delhi have higher turbidity levels.
New Delhi shows the highest bacterial contamination.
Approximately 60% of water samples are safe.
High turbidity is associated with increased bacterial contamination.
